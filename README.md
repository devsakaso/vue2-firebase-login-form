# vue2-firebase-login-form

## Before Using...
Need to install firebase and Bulma

Need to modify firebaseConfig settings on src/firebase/config.js

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
